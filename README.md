# 🏛️ Historical Places in Uzbekistan

---

### 🌟 **Loyiha haqida qisqacha**

> "O'zbekiston — bu nafaqat xarita, balki tirik tarixdir."

Ushbu veb-sayt yurtimizdagi eng qadimiy shaharlar va ularning me'moriy obidalari haqida ma'lumot beruvchi zamonaviy platformadir. Loyihaning asosiy maqsadi — **Turizmni rivojlantirish** va tariximizni **raqamli dunyoga** olib chiqish.

---

### 🛠️ **Texnologik Stack**

Loyiha eng zamonaviy va tezkor instrumentlar bilan jihozlangan:

- **Vite ⚡** — Tezkor yig'uvchi (Bundler)
- **Tailwind CSS 🎨** — Moslashuvchan va chiroyli dizayn
- **PostCSS ⚙️** — CSS-ni optimizatsiya qilish uchun
- **ESLint 🔍** — Kod sifatini nazorat qilish

### 🚀 **O'rnatish va Ishga tushirish**

Loyihani o'zingizga nusxalab olish uchun quyidagi buyruqlardan foydalaning:

1.  **Klonlash:**
    ```bash
    git clone [https://github.com/username/historical-places-uzb.git](https://github.com/username/historical-places-uzb.git)
    ```
2.  **Kutubxonalarni yuklash:**
    ```bash
    npm install
    ```
3.  **Local serverda yurgizish:**
    ```bash
    npm run dev
    ```
