# 🏛️ Historical Places in Uzbekistan

---

### 🌟 **Loyiha haqida qisqacha**

> "O'zbekiston — bu nafaqat xarita, balki tirik tarixdir."

Ushbu veb-sayt yurtimizdagi eng qadimiy shaharlar va ularning me'moriy obidalari haqida ma'lumot beruvchi zamonaviy platformadir. Loyihaning asosiy maqsadi — **Turizmni rivojlantirish** va tariximizni **raqamli dunyoga** olib chiqish.

---

### 🛠️ **Texnologik Stack**

Loyiha eng zamonaviy va tezkor instrumentlar bilan jihozlangan:

- **React.js ⚛️** — Komponentlarga asoslangan kutubxona
- **Vite ⚡** — Tezkor yig'uvchi (Bundler)
- **Tailwind CSS 🎨** — Moslashuvchan va chiroyli dizayn
- **PostCSS ⚙️** — CSS-ni optimizatsiya qilish uchun
- **ESLint 🔍** — Kod sifatini nazorat qilish

### 🚀 **O'rnatish va Ishga tushirish**

Loyihani o'zingizga nusxalab olish uchun quyidagi buyruqlardan foydalaning:

1.  **Klonlash:**
    ```bash
    git clone [https://github.com/username/historical-places-uzb.git](https://github.com/username/historical-places-uzb.git)
    ```
2.  **Kutubxonalarni yuklash:**
    ```bash
    npm install
    ```
3.  **Local serverda yurgizish:**
    ```bash
    npm run dev
    ```

### 📁 **Loyiha Strukturasi**

```bash
historical-places/
├── 📁 public/
│   └── 📁 assets/
│       └── 📁 images/  # Barcha tarixiy obidalar rasmlari
├── 📁 src/
│   ├── 📁 components/  # Qayta ishlatiladigan kichik qismlar
│   │   ├── 📄 Header.jsx  # Sayt menyusi
│   │   ├── 📄 Footer.jsx  # Pastki qism
│   │   ├── 📄 Layout.jsx  # Umumiy sahifa ko'rinishi (wrapper)
│   │   ├── 📄 PlaceCard.jsx  # Obidalar uchun kartochka
│   │   ├── 📄 ScrollObserver.jsx  # Scroll effektlari uchun
│   │   └── 📄 ScrollToTop.jsx  # Sahifani yuqoriga qaytarish
│   ├── 📁 data/
│   │   └── 📄 places.json  # Barcha obidalar haqidagi ma'lumotlar (JSON formatda)
│   ├── 📁 pages/  # Alohida sahifalar
│   │   ├── 📄 Home.jsx  # Bosh sahifa
│   │   ├── 📄 Places.jsx  # Barcha joylar ro'yxati sahifasi
│   │   └── 📄 PlaceDetail.jsx  # Bitta joy haqida batafsil ma'lumot sahifasi
│   ├── 📄 App.jsx  # Asosiy router va sahifalar ulanadigan joy
│   ├── 📄 main.jsx  # React-ni DOM-ga ulaydigan asosiy fayl
│   ├── 📄 App.css  # Global uslublar
│   └── 📄 index.css  # Tailwind CSS ulanishi
├── 📄 tailwind.config.js  # Tailwind sozlamalari
├── 📄 vite.config.js  # Vite loyiha sozlamalari
└── 📄 package.json  # Kutubxonalar va scriptlar ro'yxati
```
